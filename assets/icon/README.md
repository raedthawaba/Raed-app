# App Icon Assets

This folder should contain your app icon images.

## Required Files:

1. **app_icon.png** - Main icon (at least 1024x1024 pixels)
2. **app_icon_foreground.png** - Foreground icon for adaptive icon (at least 1024x1024 pixels)

## How to add your logo:

1. Take your logo image (the one you uploaded)
2. Save it as `app_icon.png` in this folder
3. Also save it as `app_icon_foreground.png` in this folder
4. Run: `flutter pub run flutter_launcher_icons`

## After generating icons:

The icons will be automatically generated for all platforms.
